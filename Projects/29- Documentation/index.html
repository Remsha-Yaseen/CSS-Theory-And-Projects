Introduction

CSS Grid Layout is a two-dimensional layout system for the web. It lets you content in rows and columns, and has many features that make building complex layouts easy.

Unlike Flexbox, which is largely one-dimensional, Grid is designed to handle both columns and rows simultaneously.

display: grid;
Grid Container

To get started you have to define a container element as a grid by setting its display property.

This enables a grid context for all its direct children. Use the following syntax:

.container { display: grid; }
  • Defines a grid context
  • Affects direct children only
Grid Tracks

You define the columns and rows of your grid with the grid-template-columns and grid-template-rows properties.

Tracks are the space between any two lines on the grid.

grid-template-columns: 200px 200px;
  • Columns are vertical tracks
  • Rows are horizontal tracks
Fractional Units

The fr unit allows you to set the size of a track as a fraction of the free space of the grid container.

This is much more flexible than using percentages for fluid layouts.

grid-template-columns: 1fr 2fr;
Grid Gap

The gap property (formerly grid-gap) is a shorthand for row-gap and column-gap.

It allows you to create gutters between your grid items without using margins.

gap: 20px;
  • Standardizes spacing
  • Does not add space to outer edges
  • Supported in most modern browsers
