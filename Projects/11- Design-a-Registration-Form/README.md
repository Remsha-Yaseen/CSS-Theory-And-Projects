# Responsive Registration Form

A clean, dark-themed registration form built to practice **HTML5** semantic elements and **CSS3** styling techniques. This project focuses on accessibility, form validation, and responsive design.

---


## 🚀 Features

* **Semantic HTML5:** Uses `fieldset`, `legend`, and `label` for better structure and screen-reader support.
* **Custom Inputs:** Styles text inputs, radio buttons, checkboxes, and dropdowns for a cohesive "Dark Mode" aesthetic.
* **Responsive Layout:** Uses `vw`, `vh`, and `min/max-width` to ensure the form looks great on all screen sizes.
* **UI/UX Polish:** Custom `vertical-align` for inline elements and specific attribute styling for file inputs and submit buttons.

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Attribute selectors, Box Model, Google Fonts)

## 💡 Key Concepts Learned

### 1. The Box Model
Managing `padding`, `margin`, and `border` to create breathing room.


### 2. Attribute Selectors
Targeting specific inputs like `input[type="submit"]` or `input[type="file"]` without adding unnecessary classes to the HTML.

### 3. Centering and Alignment
* Using `margin: 0 auto` and `display: block` to center the form and button.
* Using `vertical-align: middle` to align checkboxes and radio buttons with their text labels.
