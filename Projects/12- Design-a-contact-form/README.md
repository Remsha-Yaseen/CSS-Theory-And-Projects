# Custom Contact Form Project

A sleek, modern contact form built to fulfill specific functional requirements while demonstrating clean UI/UX principles. This project features a "Dark Mode" aesthetic with responsive elements and full accessibility integration.

---


## 🚀 Key Features

- **Semantic HTML5:** Structured with a dedicated `.form-container` div and accessible `<label>` associations.
- **Glassmorphism Design:** Features a dark, rounded container with subtle shadows and high-contrast typography.
- **Form Validation:** Native browser validation for `required` fields and specific `email` formats.
- **Responsive Elements:** Utilizes relative units and `box-sizing` to ensure the form behaves perfectly on all screen sizes.
- **Interactive UI:** Smooth CSS transitions on the submit button and input focus states.

## 🛠️ Requirements Fulfilled

- [x] Grouped elements within a `.form-container`.
- [x] Proper association between labels and input IDs.
- [x] Specific padding, margin, and border-radius styling.
- [x] Custom button hover effects and font scaling.

## 🎨 Tech Stack

- **HTML5:** Semantic structure and form elements.
- **CSS3:** Custom properties, transitions, and layout management.

## 💡 Technical Insight: The Box Model

For this project, I used `box-sizing: border-box;` to manage the relationship between width and padding. This ensures that the 100% width of the `input` and `textarea` elements includes their internal padding, preventing them from overflowing the parent container.
